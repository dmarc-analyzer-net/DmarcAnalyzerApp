# Architecture Decision Records (ADR)

This folder captures key technical and product architecture decisions for `DmarcAnalyzerApp`.

## Conventions

- ADR filenames are ordered and immutable (for example, `0001-some-decision.md`).
- Each ADR includes: status, date, context, decision, consequences.
- New ADRs supersede older decisions when needed. Older ADRs remain for history.
